# PROC-004 — Usabilidade e consistência visual

## Objetivo
Avaliar consistência visual e previsibilidade da interface durante uso normal.

## Requisitos cobertos
- REQ-PROD-003
- REQ-CART-001
- REQ-CART-002

## Procedimento
1. Login no sistema
2. Navegar entre Products, Cart e Checkout
3. Adicionar e remover produtos
4. Retornar à listagem de produtos

## Validações
- Labels e botões são consistentes
- Estados visuais refletem ações do usuário
- Não há elementos quebrados ou desalinhados

## Possíveis novos requisitos
- Acessibilidade (contraste, foco)
- Feedback visual para ações demoradas
